# 🎂 Carte d'Anniversaire — Android Basics with Compose

**Google Developer Training | Android Basics with Compose — Unité 1**

---

## Description

Application Android simple réalisée dans le cadre du cours **Android Basics with Compose** de Google.  
Elle affiche une carte d'anniversaire avec un message, une signature et une image de fond.

---

## Aperçu

- Image de fond avec opacité
- Message d'anniversaire centré
- Signature alignée en bas

---

## Technologies utilisées

- Kotlin
- Jetpack Compose
- Android Studio

---

## Concepts appris

- Créer un composable `@Composable`
- Utiliser `Text`, `Image`, `Box`, `Column`
- Gérer les `Modifier` (padding, fillMaxSize, align)
- Utiliser `ContentScale` et `alpha` sur une image
- Stocker les textes dans `strings.xml`
- Prévisualiser avec `@Preview`

---

## Structure du projet

```
app/src/main/
├── java/com/example/cartedeprsentation/
│   └── MainActivity.kt       # Code principal Compose
└── res/
    ├── drawable/
    │   └── androidparty.png  # Image de fond
    └── values/
        └── strings.xml       # Textes de l'application
```
